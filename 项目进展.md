# 项目进展

## 当前状态

第一版定位为 `SP 广告健康监控与决策辅助系统`。

系统不让 AI 自动执行广告动作，只做：

- 异常健康监控
- 规则判定
- AI 解释与证据整理
- 人工采纳 / 修改后采纳 / 拒绝 / 加入观察
- 处理记录与复盘

## 已确认决策

- 第一版只做 `SP 商品广告`
- 默认判断周期为 `近 14 天`
- 直接接积加开放平台 API
- 第一版数据库使用 `SQLite`
- 产品目标由人工设置
- 产品目标分类采用：
  - 测试词款
  - 放量款
  - 利润款
  - 排名承接款
  - 清库存款
  - 止损款
- 建议颗粒度：
  - 产品层承载经营目标
  - 关键词 / 搜索词层承载具体异常和建议
- 第一版页面采用 5 页：
  - 广告健康驾驶舱
  - 产品目标与规则设置
  - 异常与 AI 建议队列
  - 建议详情与溯源
  - 处理记录与复盘

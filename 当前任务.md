# 当前任务

## TASK-007：建议详情与溯源页

## 任务目标

收敛建议详情与溯源交互，让运营人员能从异常队列进入详情，看到 AI 建议、规则证据、指标快照和人工确认入口。

第一版只做：

- 异常详情抽屉或详情区
- AI 建议展示
- 规则命中原因展示
- 指标快照展示
- 原始证据 JSON 展示
- 人工处理入口

不做：

- 自动执行广告动作
- 独立复杂详情路由
- 复杂图表
- 新增后端数据模型

## 验收命令

```powershell
python scripts/smoke_frontend_suggestion_detail.py
```

## 成功标准

- smoke 脚本能运行完成。
- 输出 `status=success` 或等价成功状态。
- 点击异常有详情入口。
- 详情区展示：
  - AI 建议
  - 建议等级
  - AI 解释
  - 风险提示
  - 证据摘要
  - 规则命中原因
  - 原始证据 JSON
- 详情区有人工处理入口。
- 页面不出现自动执行广告动作入口。

## 停止条件

- 需要新增广告执行能力。
- 需要修改后端服务或模型。
- 需要改动积加 API 同步。
- 验收命令失败且无法在当前允许文件内修复。

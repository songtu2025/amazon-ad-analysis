# 当前任务

## TASK-005：人工确认与动作记录

## 任务目标

让运营人员可以对 AI 建议做人工处理，并留下可复盘记录。

第一版只做：

- 采纳
- 修改后采纳
- 拒绝
- 加入观察
- 更新异常事件状态
- 保存处理人、原因、修改动作和观察周期

不做：

- 自动执行广告动作
- 调用广告执行接口
- 前端页面
- 审批流

## 验收命令

```powershell
python scripts/smoke_manual_decision.py
```

## 成功标准

- smoke 脚本能运行完成。
- 输出 `status=success` 或等价成功状态。
- 能创建建议和人工处理记录。
- 能验证以下处理类型：
  - `adopt`
  - `adopt_with_changes`
  - `reject`
  - `observe`
- `observe` 会把异常状态更新为 `observing`。
- 其他处理会把异常状态更新为 `handled`。
- 处理记录包含操作者、原因、修改动作或观察周期。
- 处理流程不包含自动执行广告动作字段。

## 停止条件

- 需要调用广告执行接口。
- 需要新增自动执行能力。
- 需要修改规则引擎或建议生成服务。
- 需要改动前端。
- 验收命令失败且无法在当前允许文件内修复。

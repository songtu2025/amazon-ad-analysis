# 当前任务

## TASK-008：处理记录与复盘页

## 任务目标

收敛处理记录与复盘页面，让运营人员能查看人工处理记录，并对 7 天 / 14 天结果做复盘记录。

第一版只做：

- 处理记录列表
- 处理记录筛选
- 人工处理结果展示
- 复盘入口
- 复盘结果保存

不做：

- 自动执行广告动作
- 复杂绩效归因
- 独立报表系统
- 新增后端数据模型

## 验收命令

```powershell
python scripts/smoke_frontend_decision_review.py
```

## 成功标准

- smoke 脚本能运行完成。
- 输出 `status=success` 或等价成功状态。
- 页面能展示处理记录与复盘。
- 至少支持 3 个筛选：
  - 处理类型
  - 处理人
  - 异常类型或建议等级
- 能看到复盘入口。
- 能看到 7 天 / 14 天复盘周期。
- 页面不出现自动执行广告动作入口。

## 停止条件

- 需要新增广告执行能力。
- 需要修改后端服务或模型。
- 需要改动积加 API 同步。
- 验收命令失败且无法在当前允许文件内修复。

# 当前任务

## TASK-006：前端异常队列

## 任务目标

实现第一版可筛选、可点击的异常队列页面，让运营人员能看到异常事件和 AI 建议状态。

第一版只做：

- 异常列表
- 基础筛选
- 异常等级展示
- 产品目标 / 异常类型 / 状态展示
- 点击异常进入详情占位

不做：

- 自动执行广告动作
- 复杂图表
- 大屏驾驶舱重设计
- 建议详情完整页

## 验收命令

```powershell
python scripts/smoke_frontend_anomaly_queue.py
```

## 成功标准

- smoke 脚本能运行完成。
- 输出 `status=success` 或等价成功状态。
- 页面能展示异常队列。
- 至少支持 3 个筛选：
  - 异常类型
  - 状态
  - 建议等级或产品目标
- 点击异常能进入详情占位或展开详情。
- 页面不出现自动执行广告动作入口。

## 停止条件

- 需要新增广告执行能力。
- 需要改动规则引擎、建议生成服务或积加 API 同步。
- 需要重新设计后端数据模型。
- 验收命令失败且无法在当前允许文件内修复。

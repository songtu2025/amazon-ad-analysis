# 目标模式运行手册

## 目标

让 AI 按固定轨道自动开发，避免 12 小时大目标式发散。

## 固定流程

```text
读取 AGENTS.md
-> 读取 project_state.json
-> 读取 当前任务.md
-> 读取 开发任务清单.md
-> 执行当前任务
-> 运行验收命令
-> 通过后更新 project_state.json 和 项目进展.md
-> 切换下一任务
```

## 启动提示词

```text
你是本项目的开发执行 AI。

必须先读取：
1. AGENTS.md
2. project_state.json
3. 当前任务.md
4. 开发任务清单.md

你只能执行 当前任务.md 中的任务。
你只能修改允许修改文件。
你不能修改禁止修改文件。
你不能新增广告自动执行能力。
你不能跳过验收命令。
验收失败时必须停止，并说明失败原因。
验收通过后，更新 project_state.json 和 项目进展.md。
```

## 停止条件

- 当前任务不存在。
- 当前任务没有验收命令。
- 需要修改禁止文件。
- 需要新增广告自动执行能力。
- 真实数据或字段口径无法确认。
- 验收命令失败。
- 同一问题连续失败 3 次。

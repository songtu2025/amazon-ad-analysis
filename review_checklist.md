# review_checklist

## 审查目标

判断 AI 是否按当前任务完成开发，而不是自由发挥。

## 必查项

- 是否读取了 `AGENTS.md`、`project_state.json`、`当前任务.md`、`开发任务清单.md`。
- 是否只修改了当前任务允许的文件。
- 是否没有修改禁止文件。
- 是否运行了当前任务指定的验收命令。
- 验收失败时是否停止。
- 是否更新了 `project_state.json` 和 `项目进展.md`。

## TASK-002 专项检查

- 异常事件是否来自 SP 关键词 / 搜索词指标。
- 默认周期是否仍为近 14 天。
- `rule_result_json` 是否说明规则命中原因。
- `evidence_json` 是否包含关键指标证据。
- 是否没有自动执行广告动作字段。
- 是否没有改动积加 API 同步逻辑。
- 是否没有改动前端。

## 一票失败

- 出现自动执行广告动作。
- 修改 `.env`。
- 修改积加 API 客户端。
- 跳过验收命令。
- 验收失败仍标记完成。

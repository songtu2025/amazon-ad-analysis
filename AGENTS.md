# AGENTS.md

## 项目定位

本项目是 Amazon SP 广告健康监控与决策辅助系统。

AI 只允许做：
- 异常识别
- 证据整理
- 建议生成
- 可信度说明
- 人工确认后的记录与复盘

AI 不允许做：
- 自动执行广告动作
- 自动改竞价
- 自动暂停 / 开启广告
- 自动新增 / 否定关键词
- 绕过人工确认流程

## 执行规则

- 所有回复、任务清单、进度说明均使用中文。
- 遵循 KISS 原则，优先做最小可验证实现。
- 每次只执行 `当前任务.md` 中定义的一个任务。
- 修改前必须先读取：
  - `AGENTS.md`
  - `project_state.json`
  - `当前任务.md`
  - `开发任务清单.md`
- 只能修改当前任务中列出的允许文件。
- 不得修改当前任务中列出的禁止文件。
- 验收命令失败时必须停止，不能继续猜测式开发。
- 完成任务后必须更新：
  - `project_state.json`
  - `项目进展.md`

## 文档规则

- 禁止无要求创建文档。
- 如任务明确要求补文档，只写当前任务必须的最小内容。
- 能用代码、脚本、配置表达的，不写复杂冗余说明。

## 事实优先

- 如果现有代码、数据或 API 结果和预期不一致，以实际结果为准。
- 如果需求不明确，先停止并说明需要确认的问题。
- 如果发现用户判断与事实冲突，直接指出。
